\section*{Introducing Matrices via Cayley Tables}

A **Cayley table** describes the structure of a finite group by listing all possible products of its elements in a grid. This grid structure naturally motivates the concept of a **matrix**—a rectangular array of numbers or symbols governed by specific algebraic rules.

\subsection*{1. The Cayley Table as a Grid}

Consider the cyclic group of order 3, $(\mathbb{Z}_3, +)$, which consists of the elements $\{0, 1, 2\}$ under addition modulo 3. We can represent its operation using a Cayley table:

\[
\renewcommand{\arraystretch}{1.5}
\begin{array}{c|ccc}
+ & 0 & 1 & 2 \\
\hline
0 & 0 & 1 & 2 \\
1 & 1 & 2 & 0 \\
2 & 2 & 0 & 1 \\
\end{array}
\]

\subsection*{2. Extracting the Matrix Structure}

If we strip away the operational labels (the headers) and focus solely on the internal grid of outputs, we transform the Cayley table into a $3 \times 3$ **matrix**, denoted as $M$:

\[
M = \begin{pmatrix}
0 & 1 & 2 \\
1 & 2 & 0 \\
2 & 0 & 1
\end{pmatrix}
\]

In this matrix form:
\begin{itemize}
    \item Each horizontal row represents the outputs generated by a single fixed element operating on the entire set.
    \item Each vertical column represents the outputs generated by the entire set operating on a single fixed element.
    \item Any specific entry can be indexed by its row ($i$) and column ($j$), written as $M_{i,j}$.
\end{itemize}

\subsection*{3. Properties Encoded in the Matrix}

By looking at the matrix $M$, you can visually identify algebraic properties of the group:
\begin{itemize}
    \item \textbf{Closure:} Do you see any strange elements in the table that are not $0, 1, $ or $2$ ?
    \item \textbf{Identity Element:} The first row and first column both match the original ordering of the group elements, revealing that $0$ is the identity.
    \item \textbf{Commutativity (Abelian Property):} The matrix is symmetric across its main diagonal ($M_{i,j} = M_{j,i}$), which proves the operation is commutative:
    \item \textbf{Associativity:} You must check all triplets $(a\circ b) \circ c = a \circ (b \circ c)$
    \item \textbf{Inverse Element:} For every element $a$, there exists an element $b$ in the set such that $a \circ b = b \circ a = e$, where $e$ is the identity.
    \[
    M = M^T = \begin{pmatrix}
    0 & 1 & 2 \\
    1 & 2 & 0 \\
    2 & 0 & 1
    \end{pmatrix}
    \]
\end{itemize}

function checkAndDisplayMatrix(studentInput) {
    var rawText = studentInput.toString(); 
    
    // Parse all numbers sequentially out of the student's text input.
    var numbers = rawText.match(/\d+/g);
    
    // Grab our target HTML placeholders to show feedback and the matrix.
    var feedbackElement = document.getElementById("matrixFeedback");
    var displayElement = document.getElementById("matrixDisplay");
    
    // 1. Safety check: We need exactly 9 numbers.
    if (!numbers || numbers.length !== 9) {
        if (feedbackElement) feedbackElement.innerHTML = "<span style='color: red;'>Please enter exactly 9 numbers separated by commas.</span>";
        if (displayElement) displayElement.innerHTML = "";
        return false; 
    }
    
    // Map the parsed strings to actual integers
    var entries = numbers.map(Number);
    
    // 2. Restrict entries strictly to the elements 0, 1, or 2
    for (var i = 0; i < 9; i++) {
        if (entries[i] !== 0 && entries[i] !== 1 && entries[i] !== 2) {
            if (feedbackElement) feedbackElement.innerHTML = "<span style='color: red;'>Error: You can only use the numbers 0, 1, and 2.</span>";
            if (displayElement) displayElement.innerHTML = "";
            return false; 
        }
    }
    
    // 3. Construct the 3x3 matrix block
    var T = [
        [entries[0], entries[1], entries[2]], 
        [entries[3], entries[4], entries[5]], 
        [entries[6], entries[7], entries[8]]  
    ];
    
    // Generate LaTeX string to dynamically show the student's matrix on the page
    var latexMatrix = "\\[ \\begin{bmatrix} " + 
                      T[0][0] + " & " + T[0][1] + " & " + T[0][2] + " \\\\ " +
                      T[1][0] + " & " + T[1][1] + " & " + T[1][2] + " \\\\ " +
                      T[2][0] + " & " + T[2][1] + " & " + T[2][2] + 
                      " \\end{bmatrix} \\]";
                      
    if (displayElement) {
        displayElement.innerHTML = "<p><strong>Your Input Matrix:</strong></p>" + latexMatrix;
        
        // Robust check for MathJax v3 / v4
        if (window.MathJax && window.MathJax.typesetPromise) {
            window.MathJax.typesetPromise([displayElement]);
        } 
        // Fallback robust check for MathJax v2
        else if (window.MathJax && window.MathJax.Hub && window.MathJax.Hub.Queue) {
            window.MathJax.Hub.Queue(["Typeset", window.MathJax.Hub, displayElement]);
        }
    }
    
    // 4. Test for associativity: (a * b) * c === a * (b * c)
    var isAssociative = true;
    var counterexampleText = "";
    
    for (var a = 0; a < 3; a++) {
        for (var b = 0; b < 3; b++) {
            for (var c = 0; c < 3; c++) {
                var LHS = T[T[a][b]][c]; 
                var RHS = T[a][T[b][c]]; 
                if (LHS !== RHS) {
                    isAssociative = false;
                    counterexampleText = "This matrix is NOT associative. <br/>" +
                        "Counterexample: (" + a + " * " + b + ") * " + c + " = " + LHS + " but " +
                        a + " * (" + b + " * " + c + ") = " + RHS + ". Try again!";
                    break; 
                }
            }
            if (!isAssociative) break;
        }
        if (!isAssociative) break;
    }
    
    // 5. Update feedback on the screen BEFORE returning status to Ximera
    if (feedbackElement) {
        if (isAssociative) {
            feedbackElement.innerHTML = "<span style='color: green; font-weight: bold; font-size: 1.1em;'>&#10003; Good job! Your matrix is associative!</span>";
        } else {
            feedbackElement.innerHTML = "<span style='color: orange; font-weight: bold;'>" + counterexampleText + "</span>";
        }
    }
    
    return isAssociative; 
}

<script>
(function() {
    const canvas = document.getElementById('cayleyCanvas');
    const ctx = canvas.getContext('2d');
    const playPauseBtn = document.getElementById('playPauseBtn');
    const resetBtn = document.getElementById('resetBtn');
    const speedSlider = document.getElementById('speedSlider');
    const tallyList = document.getElementById('tallyList');

    // A non-associative 3x3 Cayley Table (0, 1, 2)
    // Example where (1*1)*2 != 1*(1*2)
    const table = [
        [0, 1, 2],
        [1, 0, 0], // 1*1 = 0, 1*2 = 0
        [2, 1, 1]
    ];

    // Colors representing numbers 0, 1, 2
    const elementColors = {
        0: '#e74c3c', // Red-ish
        1: '#f1c40f', // Yellow
        2: '#9b59b6'  // Purple
    };

    let isPlaying = true;
    let animationSpeed = 2; 
    let currentTriplet = { a: 0, b: 0, c: 0 };
    let phase = 0; // 0: setup/move to first lookup, 1: intermediate lookup, 2: final lookup, 3: compare/meld
    let phaseProgress = 0;
    const counterexamples = [];

    // Grid coordinates helper
    const gridOffset = { x: 50, y: 70 };
    const cellSize = 50;

    function getCellCenter(row, col) {
        return {
            x: gridOffset.x + col * cellSize + cellSize / 2,
            y: gridOffset.y + row * cellSize + cellSize / 2
        };
    }

    playPauseBtn.onclick = () => {
        isPlaying = !isPlaying;
        playPauseBtn.textContent = isPlaying ? 'Pause' : 'Play';
    };

    resetBtn.onclick = () => {
        currentTriplet = { a: 0, b: 0, c: 0 };
        phase = 0;
        phaseProgress = 0;
        counterexamples.length = 0;
        tallyList.innerHTML = 'None yet.';
    };

    function updateLogic() {
        if (!isPlaying) return;

        phaseProgress += 0.01 * animationSpeed;
        if (phaseProgress >= 1) {
            phaseProgress = 0;
            phase++;

            if (phase > 3) {
                phase = 0;
                // Move to next triplet
                currentTriplet.c++;
                if (currentTriplet.c > 2) {
                    currentTriplet.c = 0;
                    currentTriplet.b++;
                    if (currentTriplet.b > 2) {
                        currentTriplet.b = 0;
                        currentTriplet.a++;
                        if (currentTriplet.a > 2) {
                            currentTriplet.a = 0; // Loop back
                            counterexamples.length = 0;
                            tallyList.innerHTML = 'None yet (Restarted loop).';
                        }
                    }
                }
            }
        }
    }

    function lerp(start, end, amt) {
        return (1 - amt) * start + amt * end;
    }

    function draw() {
        ctx.clearRect(0, 0, canvas.width, canvas.height);

        const { a, b, c } = currentTriplet;
        
        // 1. Draw Cayley Table Structure
        ctx.strokeStyle = '#333';
        ctx.lineWidth = 2;
        ctx.font = '16px sans-serif';
        ctx.fillStyle = '#000';
        
        // Headers labels
        ctx.fillText("a \\ b", gridOffset.x - 40, gridOffset.y - 15);
        for(let i=0; i<3; i++) {
            ctx.fillText(i, gridOffset.x + i*cellSize + 20, gridOffset.y - 15);
            ctx.fillText(i, gridOffset.x - 25, gridOffset.y + i*cellSize + 30);
        }

        // Render Matrix squares
        for (let r = 0; r < 3; r++) {
            for (let col = 0; col < 3; col++) {
                ctx.fillStyle = elementColors[table[r][col]];
                ctx.fillRect(gridOffset.x + col * cellSize, gridOffset.y + r * cellSize, cellSize, cellSize);
                ctx.strokeRect(gridOffset.x + col * cellSize, gridOffset.y + r * cellSize, cellSize, cellSize);
            }
        }

        // 2. Compute Intermediate values
        const ab = table[a][b];
        const bc = table[b][c];
        const leftFinal = table[ab][c];
        const rightFinal = table[a][bc];

        // 3. Track robot positions based on timeline phase
        let blackPos = { x: 450, y: 120 };
        let bluePos = { x: 450, y: 220 };
        let mergeColor = null;

        // Path positions maps
        const pos_a_b = getCellCenter(a, b);
        const pos_ab_c = getCellCenter(ab, c);
        const pos_b_c = getCellCenter(b, c);
        const pos_a_bc = getCellCenter(a, bc);

        if (phase === 0) {
            // Moving from starts to their first lookups
            blackPos.x = lerp(450, pos_a_b.x, phaseProgress);
            blackPos.y = lerp(120, pos_a_b.y, phaseProgress);

            bluePos.x = lerp(450, pos_b_c.x, phaseProgress);
            bluePos.y = lerp(220, pos_b_c.y, phaseProgress);
        } else if (phase === 1) {
            // At first lookup point, reading results
            blackPos = pos_a_b;
            bluePos = pos_b_c;
        } else if (phase === 2) {
            // Moving to final lookups
            blackPos.x = lerp(pos_a_b.x, pos_ab_c.x, phaseProgress);
            blackPos.y = lerp(pos_a_b.y, pos_ab_c.y, phaseProgress);

            bluePos.x = lerp(pos_b_c.x, pos_a_bc.x, phaseProgress);
            bluePos.y = lerp(pos_b_c.y, pos_a_bc.y, phaseProgress);
        } else if (phase === 3) {
            // Moving to evaluation chamber to meld
            const targetX = 520;
            const targetY = 170;
            blackPos.x = lerp(pos_ab_c.x, targetX, phaseProgress);
            blackPos.y = lerp(pos_ab_c.y, targetY, phaseProgress);

            bluePos.x = lerp(pos_a_bc.x, targetX, phaseProgress);
            bluePos.y = lerp(pos_a_bc.y, targetY, phaseProgress);

            if (phaseProgress > 0.6) {
                if (leftFinal === rightFinal) {
                    mergeColor = '#2ecc71'; // Green
                } else {
                    mergeColor = '#e74c3c'; // Red
                    // Record uniquely once per step transition
                    const key = `(${a}*${b})*${c} = ${leftFinal} ≠ ${a}*(${b}*${c}) = ${rightFinal}`;
                    if (!counterexamples.includes(key)) {
                        counterexamples.push(key);
                        tallyList.innerHTML = counterexamples.join('<br>');
                    }
                }
            }
        }

        // 4. Draw HUD Text
        ctx.fillStyle = '#000';
        ctx.font = 'bold 16px sans-serif';
        ctx.fillText(`Testing Triplet: (a=${a}, b=${b}, c=${c})`, 350, 40);
        
        ctx.font = '14px sans-serif';
        ctx.fillStyle = '#111';
        ctx.fillText(`Black Box: (${a} • ${b}) • ${c} = ${phase > 0 ? ab : '?'} • ${c} = ${phase > 2 ? leftFinal : '?'}`, 350, 90);
        ctx.fillStyle = '#0066cc';
        ctx.fillText(`Blue Box: ${a} • (${b} • ${c}) = ${a} • ${phase > 0 ? bc : '?'} = ${phase > 2 ? rightFinal : '?'}`, 350, 250);

        // Evaluation Node Box
        ctx.strokeStyle = '#999';
        ctx.lineWidth = 1;
        ctx.strokeRect(480, 135, 80, 70);
        ctx.fillStyle = '#555';
        ctx.font = '12px sans-serif';
        ctx.fillText("Meld Unit", 493, 130);

        // 5. Draw the Traveling Robot boxes
        if (mergeColor) {
            ctx.fillStyle = mergeColor;
            ctx.fillRect(blackPos.x - 12, blackPos.y - 12, 24, 24);
            ctx.strokeStyle = '#000';
            ctx.strokeRect(blackPos.x - 12, blackPos.y - 12, 24, 24);
        } else {
            // Black Robot
            ctx.fillStyle = '#222';
            ctx.fillRect(blackPos.x - 10, blackPos.y - 10, 20, 20);
            // Blue Robot
            ctx.fillStyle = '#0088ff';
            ctx.fillRect(bluePos.x - 10, bluePos.y - 10, 20, 20);
        }
    }

    function loop() {
        speedSlider.oninput = () => { animationSpeed = parseInt(speedSlider.value); };
        updateLogic();
        draw();
        requestAnimationFrame(loop);
    }

    loop();
})();
</script>
