\section*{Introducing Matrices via Cayley Tables}

A **Cayley table** describes the structure of a finite group by listing all possible products of its elements in a grid. This grid structure naturally motivates the concept of a **matrix**—a rectangular array of numbers or symbols governed by specific algebraic rules.

\subsection*{1. The Cayley Table as a Grid}

Consider the cyclic group of order 3, $(\mathbb{Z}_3, +)$, which consists of the elements $\{0, 1, 2\}$ under addition modulo 3. We can represent its operation using a Cayley table:

\[
\renewcommand{\arraystretch}{1.5}
\begin{array}{c|ccc}
+ & 0 & 1 & 2 \\
\hline
0 & 0 & 1 & 2 \\
1 & 1 & 2 & 0 \\
2 & 2 & 0 & 1 \\
\end{array}
\]

\subsection*{2. Extracting the Matrix Structure}

If we strip away the operational labels (the headers) and focus solely on the internal grid of outputs, we transform the Cayley table into a $3 \times 3$ **matrix**, denoted as $M$:

\[
M = \begin{pmatrix}
0 & 1 & 2 \\
1 & 2 & 0 \\
2 & 0 & 1
\end{pmatrix}
\]

In this matrix form:
\begin{itemize}
    \item Each horizontal row represents the outputs generated by a single fixed element operating on the entire set.
    \item Each vertical column represents the outputs generated by the entire set operating on a single fixed element.
    \item Any specific entry can be indexed by its row ($i$) and column ($j$), written as $M_{i,j}$.
\end{itemize}

\subsection*{3. Properties Encoded in the Matrix}

By looking at the matrix $M$, you can visually identify algebraic properties of the group:
\begin{itemize}
    \item \textbf{Closure:} Do you see any strange elements in the table that are not $0, 1, $ or $2$ ?
    \item \textbf{Identity Element:} The first row and first column both match the original ordering of the group elements, revealing that $0$ is the identity.
    \item \textbf{Commutativity (Abelian Property):} The matrix is symmetric across its main diagonal ($M_{i,j} = M_{j,i}$), which proves the operation is commutative:
    \item \textbf{Associativity:} You must check all triplets $(a\circ b) \circ c = a \circ (b \circ c)$
    \item \textbf{Inverse Element:} For every element $a$, there exists an element $b$ in the set such that $a \circ b = b \circ a = e$, where $e$ is the identity.
    \[
    M = M^T = \begin{pmatrix}
    0 & 1 & 2 \\
    1 & 2 & 0 \\
    2 & 0 & 1
    \end{pmatrix}
    \]
\end{itemize}

\begin{javascript}
function checkAndDisplayMatrix(studentInput) {
    var rawText = studentInput.toString(); 
    
    // Parse all numbers sequentially out of the student's text input.
    var numbers = rawText.match(/\d+/g);
    
    // Grab our target HTML placeholders to show feedback and the matrix.
    var feedbackElement = document.getElementById("matrixFeedback");
    var displayElement = document.getElementById("matrixDisplay");
    
    // 1. Safety check: We need exactly 9 numbers.
    if (!numbers || numbers.length !== 9) {
        if (feedbackElement) feedbackElement.innerHTML = "<span style='color: red;'>Please enter exactly 9 numbers separated by commas.</span>";
        if (displayElement) displayElement.innerHTML = "";
        return false; 
    }
    
    // Map the parsed strings to actual integers
    var entries = numbers.map(Number);
    
    // 2. Restrict entries strictly to the elements 0, 1, or 2
    for (var i = 0; i < 9; i++) {
        if (entries[i] !== 0 && entries[i] !== 1 && entries[i] !== 2) {
            if (feedbackElement) feedbackElement.innerHTML = "<span style='color: red;'>Error: You can only use the numbers 0, 1, and 2.</span>";
            if (displayElement) displayElement.innerHTML = "";
            return false; 
        }
    }
    
    // 3. Construct the 3x3 matrix block
    var T = [
        [entries[0], entries[1], entries[2]], 
        [entries[3], entries[4], entries[5]], 
        [entries[6], entries[7], entries[8]]  
    ];
    
    // Generate LaTeX string to dynamically show the student's matrix on the page
    var latexMatrix = "\\[ \\begin{bmatrix} " + 
                      T[0][0] + " & " + T[0][1] + " & " + T[0][2] + " \\\\ " +
                      T[1][0] + " & " + T[1][1] + " & " + T[1][2] + " \\\\ " +
                      T[2][0] + " & " + T[2][1] + " & " + T[2][2] + 
                      " \\end{bmatrix} \\]";
                      
    if (displayElement) {
        displayElement.innerHTML = "<p><strong>Your Input Matrix:</strong></p>" + latexMatrix;
        
        // Robust check for MathJax v3 / v4
        if (window.MathJax && window.MathJax.typesetPromise) {
            window.MathJax.typesetPromise([displayElement]);
        } 
        // Fallback robust check for MathJax v2
        else if (window.MathJax && window.MathJax.Hub && window.MathJax.Hub.Queue) {
            window.MathJax.Hub.Queue(["Typeset", window.MathJax.Hub, displayElement]);
        }
    }
    
    // 4. Test for associativity: (a * b) * c === a * (b * c)
    var isAssociative = true;
    var counterexampleText = "";
    
    for (var a = 0; a < 3; a++) {
        for (var b = 0; b < 3; b++) {
            for (var c = 0; c < 3; c++) {
                var LHS = T[T[a][b]][c]; 
                var RHS = T[a][T[b][c]]; 
                if (LHS !== RHS) {
                    isAssociative = false;
                    counterexampleText = "This matrix is NOT associative. <br/>" +
                        "Counterexample: (" + a + " * " + b + ") * " + c + " = " + LHS + " but " +
                        a + " * (" + b + " * " + c + ") = " + RHS + ". Try again!";
                    break; 
                }
            }
            if (!isAssociative) break;
        }
        if (!isAssociative) break;
    }
    
    // 5. Update feedback on the screen BEFORE returning status to Ximera
    if (feedbackElement) {
        if (isAssociative) {
            feedbackElement.innerHTML = "<span style='color: green; font-weight: bold; font-size: 1.1em;'>&#10003; Good job! Your matrix is associative!</span>";
        } else {
            feedbackElement.innerHTML = "<span style='color: orange; font-weight: bold;'>" + counterexampleText + "</span>";
        }
    }
    
    return isAssociative; 
}
