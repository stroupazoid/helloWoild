\section*{Introducing Matrices via Cayley Tables}

A **Cayley table** describes the structure of a finite group by listing all possible products of its elements in a grid. This grid structure naturally motivates the concept of a **matrix**—a rectangular array of numbers or symbols governed by specific algebraic rules.

\subsection*{1. The Cayley Table as a Grid}

Consider the cyclic group of order 3, $(\mathbb{Z}_3, +)$, which consists of the elements $\{0, 1, 2\}$ under addition modulo 3. We can represent its operation using a Cayley table:

\[
\renewcommand{\arraystretch}{1.5}
\begin{array}{c|ccc}
+ & 0 & 1 & 2 \\
\hline
0 & 0 & 1 & 2 \\
1 & 1 & 2 & 0 \\
2 & 2 & 0 & 1 \\
\end{array}
\]

\subsection*{2. Extracting the Matrix Structure}

If we strip away the operational labels (the headers) and focus solely on the internal grid of outputs, we transform the Cayley table into a $3 \times 3$ **matrix**, denoted as $M$:

\[
M = \begin{pmatrix}
0 & 1 & 2 \\
1 & 2 & 0 \\
2 & 0 & 1
\end{pmatrix}
\]

In this matrix form:
\begin{itemize}
    \item Each horizontal row represents the outputs generated by a single fixed element operating on the entire set.
    \item Each vertical column represents the outputs generated by the entire set operating on a single fixed element.
    \item Any specific entry can be indexed by its row ($i$) and column ($j$), written as $M_{i,j}$.
\end{itemize}

\subsection*{3. Properties Encoded in the Matrix}

By looking at the matrix $M$, you can visually identify algebraic properties of the group:
\begin{itemize}
    \item \textbf{Closure:} Do you see any strange elements in the table that are not $0, 1, $ or $2$ ?
    \item \textbf{Identity Element:} The first row and first column both match the original ordering of the group elements, revealing that $0$ is the identity.
    \item \textbf{Commutativity (Abelian Property):} The matrix is symmetric across its main diagonal ($M_{i,j} = M_{j,i}$), which proves the operation is commutative:
    \item \textbf{Associativity:} You must check all triplets $(a\circ b) \circ c = a \circ (b \circ c)$
    \item \textbf{Inverse Element:} For every element $a$, there exists an element $b$ in the set such that $a \circ b = b \circ a = e$, where $e$ is the identity.
    \[
    M = M^T = \begin{pmatrix}
    0 & 1 & 2 \\
    1 & 2 & 0 \\
    2 & 0 & 1
    \end{pmatrix}
    \]
\end{itemize}

\begin{javascript}
function checkAssociativeCayley(studentInput) {
    // 1. Get the raw string input or parsed elements
    // Ximera's studentInput.evaluate() or .toString() exposes the math structure
    var rawText = studentInput.toString(); 
    
    // We expect a LaTeX matrix structure. Usually represented as:
    // "[[row1_col1, row1_col2, row1_col3], [row2_col1, ...], ...]"
    // or a string block containing LaTeX matrix format.
    // Let's parse all numbers sequentially out of the string.
    var numbers = rawText.match(/\d+/g);
    
    // Safety checks: we need exactly 9 numbers
    if (!numbers || numbers.length !== 9) {
        return false; 
    }
    
    // Map them to integers
    var entries = numbers.map(Number);
    
    // 2. Restrict entries strictly to 0, 1, or 2
    for (var i = 0; i < 9; i++) {
        if (entries[i] !== 0 && entries[i] !== 1 && entries[i] !== 2) {
            return false; 
        }
    }
    
    // 3. Map the linear array of 9 numbers to a 3x3 Cayley table matrix
    // Cayley Table T[row][col] mapping {0, 1, 2} to index {0, 1, 2}
    var T = [
        [entries[0], entries[1], entries[2]],
        [entries[3], entries[4], entries[5]],
        [entries[6], entries[7], entries[8]]
    ];
    
    // 4. Test for associativity: (a * b) * c === a * (b * c)
    // Here, operation a * b is defined by T[a][b]
    for (var a = 0; a < 3; a++) {
        for (var b = 0; b < 3; b++) {
            for (var c = 0; c < 3; c++) {
                var LHS = T[T[a][b]][c]; // (a * b) * c
                var RHS = T[a][T[b][c]]; // a * (b * c)
                if (LHS !== RHS) {
                    return false; // Found a counterexample to associativity
                }
            }
        }
    }
    
    return true; // Meets all conditions
}
\end{javascript}
